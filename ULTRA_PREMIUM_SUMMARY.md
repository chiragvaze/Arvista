## üêõ Known Issues & Solutions

### Issue: Cursor not appearing
**Solution:** Check if `initCursor()` is called. Disable on mobile in CSS.

### Issue: WebGL errors
**Solution:** Fallback is automatic. Check browser support.

### Issue: Slow performance on mobile
**Solution:** Animations auto-reduce. Check `prefers-reduced-motion`.

### Issue: Gallery images not loading
**Solution:** Check `window.artworksData` is defined before gallery init.
